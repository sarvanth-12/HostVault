⚖️
Nginx — Load Balancer

Distributes incoming HTTP requests across Flask/Waitress instances. Handles SSL termination and routes requests for high availability.

Reverse Proxy · HA
🐍
Flask + Waitress

Python backend for all API routes — authentication, file operations, user management. Waitress is the production WSGI server. Flask-Limiter protects auth endpoints.

Python · WSGI · Rate-Limited
🐘
PostgreSQL + SQLAlchemy

Stores user accounts, hashed passwords, file metadata (name, size, upload date, owner), and deletion records. All queries via SQLAlchemy ORM.

Relational DB · ORM
🗄️
MinIO — Distributed Storage

Multi-node object storage cluster that physically holds all uploaded files. S3-compatible API. Files are sharded across nodes for redundancy.

Object Storage · Multi-Node
🔒
Security Layer

Passwords hashed before storage. Flask-Limiter enforces rate limits on login and register endpoints to block brute-force. Role-based access (User / Admin).

RBAC · Hashing · Rate Limit
🗑️
Soft Delete & Recovery

Deleted files move to a trash bin with a 30-day recovery window. After expiry the object is permanently removed from MinIO and purged from PostgreSQL.

Trash Bin · 30-day TTL

Features

Everything built in

HostVault is a complete self-hosted storage solution — no third-party dependencies, no data leaving your server.

📁
File Management

Upload, download, sort and search your files. Drag-and-drop support with real-time upload progress. All files stored securely in MinIO.

🗑️
30-Day Trash Recovery

Deleted files go to trash with a 30-day recovery window. Restore anytime before expiry, or permanently delete to free up space.

🔐
Secure Authentication

Email/password login with bcrypt hashing, plus Google OAuth sign-in. Rate-limited to prevent brute force. Session-based with secure cookies.

👥
Role-Based Access

Two roles — User and Admin. Admins can manage all users, toggle accounts, change roles, and monitor storage usage across the entire system.

📊
Storage Quotas

Each user has a configurable storage quota (default 10 GB). Real-time usage bar in the dashboard. Admins can adjust per-user limits anytime.

📋
Activity Logs

Every action is logged — uploads, downloads, logins, deletions. Admins can filter by event type or search by user to audit all activity.
